# Работа со строками

- Цель урока:
  Вы научитесь осуществлять базовые операции над строками. Например, конкатенацию
  (объединение) строк, обрезку строки, поиск нужных символов и фраз.

- Содержание урока:

1. Конкатенация строк
2. Спецсимволы
3. Доступ к символам и длина строки
4. Методы строк

- Дополнительные материалы:
  [Все методы строк в JavaScript](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String)
  [Список спецсимволов](https://learn.javascript.ru/string#spetssimvoly)

В JavaScript есть 2 основных способа конкатенации (объединения) строк. Первый
реализуется с помощью оператора +, второй - с помощью следующей конструкции `${}`.

- Примеры:
  const name = 'Максим';
  const job = 'Front-End разработчик';
  const result1 = name + ' ' + job;
  const result2 = `${name} ${job}`;

Из этих 2-х способов рекомендуется использовать 2-й, потому что он более удобен и
благодаря ему код становится более сжатым.

# Используемые методы в этом уроке

1. - .trim() - Удаляет пробелы с обеих сторон;
2. - .toUpperCase() - Возвращает значение строки, на которой он был вызван, преобразованное в верхний регистр;
3. - .toLowerCase() - Возвращает значение строки, на которой он был вызван, преобразованное в нижний регистр;
4. - .length - Устанавливает или возвращает число элементов массива (Длиина массива);
5. - .repeat() - Конструирует и возвращает новую строку, содержащую указанное количество соединённых вместе копий строки,
     на которой он был вызван.

### Замена символов в строке

1. - .replace() - Возвращает новую строку с заменой указаного слова или выражения на другое указаное;
2. - .replaceAll() - Возвращает новую строку с заменой всех указанных слов или выраженй на другие указаные;

### Найти символы

1. - .indexOf() - возвращает первый индекс, по которому данный элемент может быть найден в массиве или -1, если такого индекса нет.
2. - .includes() - определяет, содержит ли массив определённый элемент, возвращая в зависимости от этого true или false.

### Обрезка строки

1. - .slice() - Извлекает часть строки и возвращает новую строку без изменения оригинальной строки (Обрезает с, по);
2. - .substring() - Возвращает подстроку строки между двумя индексами, или от одного индекса и до конца строки. (раюотает так же как и slice());
