# Debug

- Цель урока:
Вы научитесь находить ошибки (баги) в коде разными способами и исправлять их.

- Содержание урока:
1. Самый популярный способ дебага - console.log.
2. Дебаг с помощью breakpoints в VS Code и в Google Chrome.
